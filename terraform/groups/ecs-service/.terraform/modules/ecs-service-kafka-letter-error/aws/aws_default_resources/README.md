# aws_default_resources

> This project was generated by [generator-tf-module](https://github.com/sudokar/generator-tf-module)

## Overview

A module that takes ownership of VPC default resources and makes them secure by removing any routes from the route table and egress rules from the security group.

## Usage

```hcl
module "aws_default_resources" {
  source = "git@github.com:companieshouse/terraform-modules//aws/aws_default_resources"

  region = eu-west-2
  azs    = ["a", "b", "c"]
}

```
